🤖 How the AI predicts

Three machine-learning models — Random Forest, Gradient Boosting, and XGBoost — get blended into one prediction. Each looks at a different time horizon: the last 3 draws (recent form), the last 22 draws' average and spread (medium-term trend), and how overdue each number is based on its gap since it last appeared (long-term memory).

Daily Grand is a fair, independent draw, so no model can find a real edge — the scorecard below proves it honestly. Consider this a fun extra, not a strategy.
